# Welcone to Twenty-One (Terminal BlackJack)
## This program is a simulation of Black Jack where it tells you your safety percentage of hitting/dealing a card.


## Requirements:
  1. An IDE that can run C++ 17 or higher

## Instructions:
  1. Download the "Game Folder"
  2. Run "TwentyOne.cpp" in your IDE
  3. Have fun!
  
  
## Controls: 
  * Input:
  - 'r' for rules
  - 'p to play
  - 'h' to hit
  - 's' to stand.

## Tip:
When you are playing the rounds, give the program about 5 seconds to load up the next round so you can view the results of your last played round.

## Warning: 
The "split" function has not been implemented yet as the game is in a Work Of Progress.
